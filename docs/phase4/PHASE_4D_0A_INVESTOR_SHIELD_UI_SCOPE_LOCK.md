## Purpose

## Core UI Principle
The UI must not make advisory items appear authoritative, and it must not visually soften blocked or rejected movement.

Deterministic governance, capital protection, True MAO, and deal classification must remain visually dominant over advisory content.

- Investor Shield gates

- missing evidence warnings
- blocked movement warnings
- manual review indicators
- task recommendations
- protected movement explanations
- waiver visibility and waiver reason display
- deterministic governance dominance
