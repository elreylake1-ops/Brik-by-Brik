# Phase 4D-0D Investor Shield Warning Copy

## Purpose
This document defines future UI copy guidance for Investor Shield warnings and blocked movement, but adds no UI code.

## Copy Principles
- clear and operational
- no scare-copy unless severity is `BLOCKER` or `FATAL`
- explain what is missing
- explain next action
- preserve deterministic governance dominance
- do not imply AI/advisory evidence is hard proof

## Protected Movement Blocked Copy
Title:  
Investor Shield blocked this movement

Body:  
This deal cannot move to `{requestedStage}` yet because required due diligence gates are not clear.

Details to show:
- blocking gate names
- missing evidence
- required next action
- reminder that blocked movement did not change the pipeline state

## Needs Review Copy
Title:  
Investor Shield review required

Body:  
This deal has caution or incomplete due diligence items that should be reviewed before moving to `{requestedStage}`.

Details to show:
- caution gate names
- weak or missing evidence
- recommended tasks
- manual review note

## Missing Evidence Copy
Title:  
Evidence needed

Body:  
Investor Shield requires additional evidence before this gate can be treated as clear.

Details to show:
- gate name
- required evidence type
- recommended next action

## Advisory Evidence Copy
Title:  
Advisory evidence only

Body:  
This evidence may support review, but it cannot satisfy hard due diligence requirements by itself.

Specific note:  
AI visual review is advisory-only and cannot replace builder, professional, document, or measurement evidence.

## Manual Review / Waiver Copy
Title:  
Manual review required

Body:  
This gate has been waived or requires a manual decision. A clear reason must be recorded before the risk can be treated as reviewed.

Details to show:
- gate name
- waiver reason if present
- missing reason warning if absent

## Deterministic Governance Dominance Copy
Title:  
Deterministic decision remains dominant

Body:  
Investor Shield cannot override a deterministic NO-GO, True MAO breach, capital protection failure, or governance-risk classification.

## Task Recommendation Copy
- task title should be action-oriented
- reason should identify the gate
- priority should follow severity
- task copy should not imply risk is resolved until evidence is provided
