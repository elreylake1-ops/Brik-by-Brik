# Phase 2 Screenshot Placeholders

Place manual review screenshots for the client package in this folder.

Recommended files:

- `01-strong-clean-deal.png`
- `02-marginal-thin-margin.png`
- `03-no-go-negative-profit.png`
- `04-heavy-refurb-exposure.png`
- `05-unsupported-scope-warning.png`
- `06-governance-dangerous-deal.png`

What each screenshot should demonstrate:

- `01-strong-clean-deal.png`: Strong clean deal with a clear pass outcome.
- `02-marginal-thin-margin.png`: Thin-margin case presented as cautionary, not strong.
- `03-no-go-negative-profit.png`: Negative-profit or fatal downside case blocked as `NO_DEAL`.
- `04-heavy-refurb-exposure.png`: Heavy refurb case escalated for review because of capex or scope exposure.
- `05-unsupported-scope-warning.png`: Scope-confidence or verify-scope warning visible on screen.
- `06-governance-dangerous-deal.png`: Governance explanation visible for a blocked or review-required dangerous deal.

Use this folder only for review images intended to support the Phase 2 client walkthrough.
